\e[0m USEFUL COMMANDS \e[37m
cat     - show file content         mkdir   - create directory
clear   - clear the screen          proc    - process utility
echo    - print text to stdout      rim     - start text editor
kb      - set keyboard layout       rm      - remove file or dir
less    - show file content         sys     - show time, mem, disk
ls      - list the current dir      wc      - count words in file

all commands are in /bin directory

\e[0m OLIVINE LANGUAGE \e[37m
cd      - change dir                .      - execute file
eval    - calculate expression      set    - set variable

Some examples:

\e[95mFUNC\e[0m double                         \e[95mFOR\e[0m i \e[92m!(\e[96mrange\e[0m 10\e[92m)
    \e[95mRETURN \e[92m!(\e[96meval\e[0m \e[93m!0\e[0m x 2\e[92m)               \e[94mecho \e[93m!i
\e[95mEND                                 END
